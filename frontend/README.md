## ğŸš€ Instalasi & Menjalankan Server

Pastikan NodeJS sudah terinstal di komputer Anda. Ikuti langkah berikut untuk menjalankan localhost frontend:

### 1. Masuk ke Direktori & Install Dependencies
Buka terminal/command prompt, lalu jalankan perintah berikut:

```bash
cd .\frontend\
npm install
```
### 2. Jalankan Web Frontend
Setelah instalasi selesai, jalankan frontend dengan perintah:

```bash
npm run dev
```
Web frontend akan aktif melalui localhost.
